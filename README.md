# 钢丝绳报价管理系统

《钢丝绳报价管理系统》是根据钢丝绳生产企业开发的一款报价的系统，方便查询与客户的经营情况。 

系统包含： 1、基础设置：客户信息、产品信息、加工方式、提醒天数、短尺项、单位信息 2、报价管理：报价登记、报价状态 3、报价查询：报价到期、期间询价、期间成单、报价明细 

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe(或Hadp.exe)"文件即可完整运行对应的信息管理系统。
